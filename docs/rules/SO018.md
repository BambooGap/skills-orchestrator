# SO018 Missing License

`builtin/engineering-grade` requires every skill to declare license metadata.

## Why It Matters

Skills are executable instructions. A repository cannot safely import or
redistribute them without knowing the license attached to the instruction text.

## Fix

Add an SPDX license identifier to the skill frontmatter:

```yaml
license: MIT
```

The built-in engineering-grade allowlist accepts `MIT` and `Apache-2.0`.
