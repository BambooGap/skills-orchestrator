# SO014: missing-review-window

Severity: warning

`builtin/engineering-grade` expects each shared skill to declare:

- `reviewed_at`
- `expires_at`

These fields let platform teams tell when a skill was last reviewed and when it must be reviewed
again.

## Fix

```yaml
reviewed_at: 2026-06-21
expires_at: 2026-12-21
```
