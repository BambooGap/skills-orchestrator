# SO020 Missing Import Provenance

`builtin/engineering-grade` requires externally sourced skills to carry import
provenance.

## Why It Matters

An external skill should be traceable back to the observed source URL, resolved
ref, immutable commit, fetched content hash, and fetch timestamp. Frontmatter
inside the imported file is not trusted as provenance by itself.

## Fix

Record provenance for skills whose `source` is an HTTP(S) URL:

```yaml
provenance:
  source_url: https://raw.githubusercontent.com/example/repo/0123456789abcdef0123456789abcdef01234567/skill.md
  source_ref: main
  source_commit: 0123456789abcdef0123456789abcdef01234567
  content_hash: sha256:...
  fetched_at: 2026-06-21T00:00:00Z
```

The `skills-orchestrator import` command writes this metadata automatically for
GitHub imports.
