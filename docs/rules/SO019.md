# SO019 Disallowed License

`builtin/engineering-grade` only accepts approved skill licenses.

## Why It Matters

Instruction supply-chain checks should fail closed. Unknown, missing, or
copyleft licenses can be legitimate, but they need an explicit organization
policy decision before being used as runtime instructions.

## Fix

Use one of the built-in allowed SPDX identifiers:

```yaml
license: MIT
```

or

```yaml
license: Apache-2.0
```

For organization-specific allowlists, use a local declarative policy pack.
