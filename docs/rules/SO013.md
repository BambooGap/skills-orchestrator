# SO013: invalid-load-policy

Severity: error

Skill `load_policy` metadata must be one of the supported skill policies.

## Why It Matters

Invalid load policies make runtime injection and CI policy decisions ambiguous. This finding points
at the offending skill metadata rather than treating the whole config as an opaque fatal error.

## Fix

Use one of:

```yaml
load_policy: free
```

or:

```yaml
load_policy: require
```
