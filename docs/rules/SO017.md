# SO017: declarative-policy-violation

Severity: warning by default; local policy packs may emit `error`, `warning`, or `info`.

A local declarative policy pack was not satisfied. Declarative packs are YAML/JSON data contracts;
they do not execute Python code from the target repository.

## Fix

Inspect the diagnostic metadata for:

- `policy_pack`
- `declarative_rule`
- `field`

Then update the skill metadata or the local policy pack rule.
